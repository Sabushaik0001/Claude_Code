# Claude Code Slash Commands -- Complete Advanced README

This document covers ALL Claude Code slash commands including:

Basic Commands: - /usage - /stats - /status - /rename - /clear -
/compact - /context

Advanced Commands: - /memory - /model - /permissions - /plan - /resume -
/help - /agents

Expert Commands: - /config - /cost - /tools - /sessions

Each command includes:

-   What it is
-   Why it is used
-   When to use it
-   Example usage
-   Output explanation

------------------------------------------------------------------------

# /usage

## What

Shows overall usage metrics.

## Why

Track token usage and productivity.

## When

Monitor cost and performance.

## Example

    /usage
